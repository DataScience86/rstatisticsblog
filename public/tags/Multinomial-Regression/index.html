    February 22, 2020
    R Code, Case Study, Multinomial Regression

    Multinomial logistic regression is used when the target variable is categorical with more than two levels. It is an extension of binomial logistic regression. Overview - Multinomial Regression Multinomial regression is used to predict the nominal target variable. In case the target variable is of ordinal type, then we need to use ordinal logistic regression. In this tutorial, we will see how we can run multinomial logistic regression.
